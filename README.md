Искал себя в тех выходных
Два через два
Нам бы выпить яду на двоих
Пасть до утра
В пасть демона
Нам страсть не дана
Напасть заперла
Нам класть на себя, но
Ищу себя в подъездах с кем-то два, три часа
Нам бы выпилиться этой ночью
Впасть навсегда по фазе сна
Съесть феназепам
Нас давит весна
Но мне по кайфу суета
